Famous Quotes Bot

Are you looking for a quote? Famous Quotes Bot is here to the rescue! Search quotes filtered by author or genre with simple commands

Commands

Server Prefix: ~

  • help

    Shows help about the bot, a command, or a category.

  • [quote|qt|random] [tags] [author]

    Sends a quote by author and/or tags.

  • tags

    Sends a list of all tags available.

  • about

    Tells information about the bot itself.

  • join

    Sends a link to add Famous Quotes Bot to your server.

  • uptime

    Check the bot total uptime.

  • version

    Tells the version of the bot.
